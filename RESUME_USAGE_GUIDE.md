## Best Practices for MNC-Standard Resumes

### Content Guidelines

**1. Use Strong Action Verbs**
Start each bullet point with powerful verbs:
- ‚úÖ "Developed", "Achieved", "Led", "Optimized"
- ‚ùå "Responsible for", "Worked on", "Helped with"

**2. Quantify Everything**
Include numbers, percentages, and metrics:
- ‚úÖ "Increased sales by 35% in Q1 2023"
- ‚ùå "Improved sales performance"

**3. Show Impact**
Demonstrate the value you brought:
- ‚úÖ "Reduced deployment time by 60% through CI/CD implementation"
- ‚ùå "Implemented CI/CD pipelines"

**4. Keep It Concise**
- Aim for 300-800 words (1-2 pages)
- Use bullet points, not paragraphs
- Remove outdated or irrelevant experience

**5. Tailor to the Role**
- Highlight relevant skills and experience
- Use keywords from the job description
- Emphasize achievements related to the target position

### Structure Guidelines

**Essential Sections (in order):**
1. Contact Information (top)
2. Professional Summary (3-4 lines)
3. Core Competencies/Skills
4. Professional Experience (reverse chronological)
5. Education
6. Certifications (if applicable)
7. Technical Skills (for tech roles)

**Optional Sections:**
- Projects (especially for developers)
- Awards & Recognition
- Publications
- Volunteer Work
- Languages

### Formatting Guidelines

**Do:**
- Use consistent formatting throughout
- Choose professional fonts (Arial, Calibri, Times New Roman)
- Use 10-12pt font size
- Include adequate white space
- Use bullet points for readability
- Bold section headers

**Don't:**
- Use multiple font styles
- Include photos (unless required in your region)
- Use colors excessively
- Include personal information (age, marital status, religion)
- Use first-person pronouns ("I", "me", "my")

## Common Issues & Solutions

### Analysis Issues

**Issue: "Resume text is too short or empty"**
- **Solution**: Ensure your PDF/DOCX is not image-based or scanned
- Try converting to plain text first
- Check file isn't corrupted

**Issue: "File too large"**
- **Solution**: Compress your PDF or remove embedded images
- Maximum size is 5MB

**Issue: Low score despite good resume**
- **Solution**: The analyzer looks for specific elements:
  - Add more quantifiable achievements
  - Use more action verbs
  - Ensure all essential sections are present
  - Check word count (aim for 300-800)

### Generation Issues

**Issue: Missing required fields error**
- **Solution**: Fill in all fields marked with asterisk (*)
- Check that fields aren't just whitespace

**Issue: Template has too many placeholders**
- **Solution**: Fill in more optional fields before generating
- You can always edit the downloaded file later
